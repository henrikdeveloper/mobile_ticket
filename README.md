## Queue Management System (SCA)

Queue management system for medical laboratories, developed using Ionic and React.

## About the Project

This application manages the flow of patient service in a medical laboratory, controlling ticket issuance, calls by service desk, and statistical reporting. The system operates with three types of priorities and specific service rules.

## Technologies

- **Framework: Ionic + React**
- **Language: TypeScript**
- **Database: SQLite (local storage)**
- **Styling: CSS Modules + Ionic Components**
